real_vs_fake_2 (Stimuli used for Hwasung Medi-Science University Experiment)

Stimulus set and metadata for the study on human discrimination of real versus AI-generated face portraits.

Real portraits

Real portraits were sampled from the Flickr-Faces-HQ (FFHQ) dataset (Karras et al., 2019). According to the FFHQ documentation, each image retains its original Flickr license and attribution metadata, and users should comply with the license terms associated with each image. The FFHQ dataset package (metadata, scripts, and documentation) is distributed separately by NVIDIA.

Modifications

FFHQ-derived real portraits in this repository were based on images already aligned/cropped in FFHQ. Additional modifications made for the present experiment are listed in the metadata table, including the addition of a white square box at the lower right and resizing for experiment/repository use.
